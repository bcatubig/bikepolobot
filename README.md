# bikepolobot

Python Used: `3.11`

## Instructions

1. Create a python virtual env

```
python -m venv venv
```

2. Activate the venv

```
source venv/bin/activate
```

3. Install requirements

```
# activate venv
pip install -r requirements.txt
```

4. Run the bot

```
DISCORD_TOKEN='CHANGEME' python ./bot.py
```
